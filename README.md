# Virtualenv
$ pip install virtualenv
$ virtualenv venv

on `Windows`
$ venv\Scripts\activate
on `mac os` and `linux`
$ source venv/bin/activate


# Require
```
$ pip install -r requirements.txt
```

```
$ python Flask_test.py
```
